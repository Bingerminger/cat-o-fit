# Markenhinweise

Cat-O-Fit nennt an verschiedenen Stellen Produkt- und Markennamen Dritter. Diese
Nennungen erfolgen ausschließlich **beschreibend** (sogenannte lautere bzw.
nominative Nutzung), um Funktionen, Kompatibilität oder Trainingsmethoden zu
erklären. Mit den jeweiligen Inhabern besteht **keine** Verbindung, Partnerschaft,
Förderung oder Billigung.

Alle genannten Marken sind Eigentum ihrer jeweiligen Inhaber:

| Marke | Inhaber (sinngemäß) | Verwendung in Cat-O-Fit |
|---|---|---|
| **VDOT** | Jack Daniels / Run S.M.A.R.T. Project | Bezeichnung der Pace-/Form-Schätzmethode |
| **Hyrox** | HYROX GmbH | Auswählbarer Wettkampf-Typ für die Planerstellung |
| **Apple Health** | Apple Inc. | Import von Gesundheitsdaten (Datei-Export der Nutzerin/des Nutzers) |
| **Garmin** | Garmin Ltd. | Beschreibung kompatibler Aktivitäts-Exporte (GPX/TCX) |
| **Strava** | Strava, Inc. | Beschreibung kompatibler Aktivitäts-Exporte (GPX/TCX) |
| **Synology** | Synology Inc. | Empfohlene Betriebsumgebung (Web Station) |
| **iPhone**, **iPad** | Apple Inc. | Primäre Zielgeräte der PWA |

## Hinweise

- Cat-O-Fit ist ein unabhängiges, quelloffenes Hobby-/Community-Projekt.
- Die App ruft **keine** proprietären Schnittstellen dieser Anbieter auf. Importe
  (z. B. Apple Health, GPX/TCX) erfolgen über vom Nutzer selbst exportierte
  Standard-Dateien.
- „VDOT" wird als etablierter Fachbegriff der Trainingslehre verwendet; die
  zugrunde liegenden Formeln sind frei publiziert (siehe [CREDITS.md](CREDITS.md)).
- Sollte ein Markeninhaber eine Nennung beanstanden, wird sie auf Hinweis hin
  angepasst oder entfernt. Kontakt über die [SECURITY.md](SECURITY.md) bzw. die
  Repository-Issues.

Der Name **„Cat-O-Fit"** sowie der technische Projektname **„catofit"** sind
Projektbezeichnungen dieses Repositorys und nicht als eingetragene Marke
beansprucht.
